## How to install
* in MonoDevelop `open menu -> Tools -> Options`
* navigate to -> `Text Editor -> Syntax Highlighing`
* press `Add` button
* navigate to file `TomorowNight.json`
* press `Open`
* select `TomorrowNight` theme and then press `OK`
* close and reopen all your file tabs
